## Dependency

> **Warning**
>  
> You need following repository `public-bytemc`

Repository: 
```xml
<repository>
    <id>public-bytemc</id>
    <url>https://artifactory.bytemc.de/artifactory/public-bytemc</url>
</repository>
```
Dependency:
```xml
<dependency>
    <groupId>net.bytemc</groupId>
    <artifactId>evelon</artifactId>
    <version>VERSION</version>
</dependency>
```
